Sobre el proyecto

Esta PokéDex ha sido desarrollada como proyecto de evaluación de los resultados de aprendizaje RA04 y RA05. Aplica HTML5, CSS3 y JavaScript para consumir datos en tiempo real de la PokéAPI.

Incluye un buscador de Pokémon por nombre o número, un listado paginado con filtro por tipo, y acceso a múltiples endpoints de la API.

Tecnologías utilizadas

  • HTML5 — estructura semántica
  • CSS3 — estilos y animaciones
  • JavaScript — lógica y consumo de API
  • PokéAPI — fuente de datos externa
